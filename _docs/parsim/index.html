    Parsim - Particle-based Simulation

    Parsim - Particle-based Simulation

    ParSim is a fork of the Uintah simulation software with a focus on solid mechanics modeling and simulation. Current development is focussed on Vaango, a MPM-CFD code, and GranularSim, a DEM-SPH-Peridynamics code. We are also exploring the development of a user interface for these tools.
